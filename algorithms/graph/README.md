## 📚 算法列表

### 1. Tarjan算法 - 强连通分量
- **文档**: [`Tarjan算法详解.md`](./Tarjan算法详解.md)
- **代码**: [`tarjan_template.cpp`](./tarjan_template.cpp)
- **时间复杂度**: O(V+E)
- **应用场景**: 强连通分量求解、差分约束系统、2-SAT问题

## 🎯 使用指南

### 编译运行
```bash
# 编译Tarjan模板
g++ -o tarjan_test tarjan_template.cpp -std=c++17

# 运行测试
./tarjan_test
```
