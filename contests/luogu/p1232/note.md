# P1232 树的计数 - 算法题目说明

## 📚 题目描述
给定一棵有根树的DFS序列和BFS序列，求所有满足这两个序列的有根树的高度平均值。

## 🎯 解题要点
- **核心思想**：BFS序列的分层方式与树的高度直接相关
- **关键约束**：DFS序列和BFS序列的兼容性条件
- **算法复杂度**：O(n) 线性时间算法
